# DevQuiz

DevQuiz is a platform for developers to test their knowledge, challenge peers, and learn new topics through AI-generated quizzes.

## Tech Stack
-   **Frontend**: React (Vite), Tailwind CSS
-   **Backend**: Node.js, Express
-   **Database**: MongoDB
-   **Auth**: Google OAuth 2.0
-   **AI**: Gemini API

## Getting Started

1.  **Clone the repository**
2.  **Install Backend Dependencies**
    ```bash
    cd backend
    npm install
    ```
3.  **Install Frontend Dependencies**
    ```bash
    cd frontend
    npm install
    ```
4.  **Environment Setup**
    Create `backend/.env` based on the template provided in the source code or walkthrough.
5.  **Run the App**
    -   Backend: `npm run dev` (in backend folder)
    -   Frontend: `npm run dev` (in frontend folder)

## Features
-   Google Login
-   Create/Join Groups
-   Generate Quizzes with AI
-   Real-time Scoring
-   Leaderboards
